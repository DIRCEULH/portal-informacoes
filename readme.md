📄 Descrição do Projeto

Este projeto consiste no desenvolvimento de um portal corporativo para centralização e gestão de arquivos e informações internas da empresa, utilizando a plataforma WordPress e linguagem PHP.

O sistema foi projetado para facilitar o acesso, organização e compartilhamento de documentos, comunicados e conteúdos institucionais, proporcionando um ambiente seguro, intuitivo e acessível para colaboradores.

🏢 Objetivo

Criar uma plataforma digital interna onde a empresa possa:

Centralizar documentos importantes
Compartilhar informações com colaboradores
Organizar conteúdos por setores ou categorias
Melhorar a comunicação interna
🚀 Funcionalidades
📁 Gestão de arquivos (upload, download e organização)
👥 Controle de acesso por usuários (login restrito)
📰 Publicação de notícias e comunicados internos
🗂️ Organização por categorias e departamentos
🔍 Sistema de busca de conteúdos
📱 Interface responsiva (acesso via desktop e mobile)
🔒 Área restrita para colaboradores
🛠️ Tecnologias Utilizadas
WordPress
PHP
MySQL
HTML5 & CSS3
JavaScript
